classmethod from_harte_chord_string(chord_string)[source]¶

Create a chord from a chord string in the syntax proposed in the 2005 paper ‘SYMBOLIC REPRESENTATION OF MUSICAL CHORDS: A PROPOSED SYNTAX FOR TEXT ANNOTATIONS’ by Harte et al.

Parameters:chord_string – Chord string in syntax proposed by Harte et al.
Returns:Chord type as specified by the chord string

True

Create a chord from a root note, shorthand string, degree string list, bass degree and optional shorthand type

Parameters:
  • root_note – Root note of the chord
  • shorthand_string – Shorthand string that specifies the degrees (may be empty)
  • degree_string_list – List of degrees. A degree starting with * means that this degree is not in the chord
  • bass_degree – Bass degree Interval
  • shorthand_type – String that specifies the shorthand type: either ‘harte’ or ‘tab’
Returns:

Chord as specified by the root note, shorthand, degrees and bass
